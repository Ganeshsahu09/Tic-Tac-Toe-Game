🎮 Tic-Tac-Toe Game

A simple and interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript. This project features a clean user interface, smooth gameplay, sound effects, and a winning popup modal for a better user experience.

✨ Features
Two-player mode (X vs O)
Interactive and responsive UI
Turn indicator display
Win detection system
Popup modal on game win
Reset and Play Again functionality
Sound effects for moves and victory
Background image and animations

🛠️ Technologies Used
HTML5 for structure
CSS3 for styling and animations
JavaScript (ES6) for game logic

🚀 How to Play

Open index.html in your browser.
Player X starts the game.
Click on any empty box to place your mark.
The first player to align three symbols in a row, column, or diagonal wins.
Click Reset or Play Again to start a new game.
